## / (Root)

- **docusaurus.config.ts**: Main configuration file.
- **sidebars.ts**: Sidebar navigation structure.
- **src/**: Source code for React components and styles.
  - **components/**: Custom React components (Homepage, Features, etc.).
  - **css/**: Global styles (custom.css).
  - **pages/**: Standalone pages (index.tsx, showcase.tsx).
- **docs/**: Main documentation files (Markdown/MDX).
  - **intro.md**: Introduction page.
  - **tutorial-basics/**: Example tutorial section.
- **blog/**: Blog posts (optional).
- **static/**: Static assets (images, fonts, global files).
  - **img/**: Images.
- **i18n/**: Translations (if enabled).
- **versioned_docs/**: Versioned documentation.
- **versioned_sidebars/**: Versioned sidebars.
